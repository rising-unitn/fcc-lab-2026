* General information
- This repository contains the code for the lab exercises of the course. Each lab exercise is contained in a separate folder (e.g., =e000=, =e001=, etc.) and includes a =README.org= file with instructions on how to complete the exercise.
- If the exercise folder contains only the =README.org=, you should just follow those instructions, while if it contains other files, then you need to =cd= inside the folder and follow the steps from there as they are relative to the specific exercise.
- Lab exercises are meant to work with the environment students are given access to (as explained in the =Lab00= slide pack). They may work on other environments (e.g., your laptop) but I cannot guarantee the full compatibility.
